本手册面向 **桌面版（Electron）普通用户**，重点说明“在界面里如何完成操作”。我们假定你主要通过侧边栏点击进入功能页面，不需要理解代码或接口。

## 你能在界面里完成的事情

- 连接手机设备（含 USB、WiFi、二维码配对等）并查看在线状态
- 通过“对话”让 AI 执行任务
- 保存常用任务为 Workflow，并一键复用
- 设定定时任务，让 Workflow 自动运行
- 查看任务历史记录与详情
- 在桌面版查看日志文件

## 如何阅读本手册

1. 如果是首次使用，先看「快速开始」。
2. 然后按你要用的功能阅读「功能说明」。
3. 如果遇到问题，看「问题排查」和「FAQ」。
