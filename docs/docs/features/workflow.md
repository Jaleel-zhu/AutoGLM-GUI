---
title: Workflow 工作流
---

Workflow 用于保存常用任务文本，便于一键复用。

## 页面入口

左侧导航栏：**「Workflows」**

## 主要按钮与文本

- 「新建 Workflow」
- 「创建 Workflow」
- 「编辑 Workflow」
- 「确认删除此 Workflow？」
- 「快速执行」
- 「选择 Workflow」

## 使用流程

1. 点击「新建 Workflow」
2. 输入「名称」与「任务内容」
3. 点击「创建 Workflow」保存
4. 需要修改时点击「编辑 Workflow」
5. 删除时会弹出「确认删除此 Workflow？」

## 快速执行

在对话页面中可使用「快速执行」入口，选择「选择 Workflow」后快速填入任务内容。
