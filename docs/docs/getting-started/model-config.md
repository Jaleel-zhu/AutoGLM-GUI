---
title: 模型配置（界面版）
---

模型配置在「对话」页面完成，所有字段与按钮与界面保持一致。

## 配置入口

- 按钮：**「配置」**
- 保存按钮：**「保存配置」**
- 取消按钮：**「取消」**

## 必填字段（基础模型）

- **「Base URL」**
- **「API Key」**
- **「模型名称」**

## 分层代理相关字段

当你使用分层代理时，需要填写决策模型字段：

- **「决策模型 Base URL」**
- **「决策模型 API Key」**
- **「决策模型名称」**

界面上还会出现以下标签：

- 「视觉模型」
- 「决策模型」
- 「选择预设配置」
- 「选择决策模型预设」
