---
title: 首次使用流程
---

以下步骤完全对应桌面版界面上的按钮与文字。

## 1. 打开应用

左侧导航栏入口为：

- 对话
- Workflows
- 历史记录
- 定时任务
- 日志

## 2. 连接设备

进入「对话」页面后，左侧设备列表标题为「设备」。

- 若未连接设备，会显示「未连接设备」
- 可点击「添加无线设备」开始连接流程

## 3. 配置模型

在对话页点击「配置」，进入设置弹窗，主要按钮与字段如下：

- 按钮：
  - 「保存配置」
  - 「取消」
- 字段：
  - 「Base URL」
  - 「API Key」
  - 「模型名称」

未配置时界面会提示「请先配置您的 API 设置」。

## 4. 开始任务

选择设备后，在输入框中输入任务：

- 输入框提示：
  - 「发送消息...」（对话输入）
  - 或「您想做什么？（Cmd+Enter 发送）」

点击发送即可开始执行。
